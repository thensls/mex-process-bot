# SOPs (Standard Operating Procedures)

This directory contains step-by-step procedures for common MEX scenarios.

## Naming Convention
`kebab-case-description.md` — e.g., `member-status-change.md`, `event-cancellation.md`

## Format
Each SOP should include:
1. **Title** — What this SOP covers
2. **When to use** — Trigger conditions
3. **Steps** — Numbered walkthrough
4. **Common mistakes** — What to watch out for
5. **Escalation** — When this SOP isn't enough
