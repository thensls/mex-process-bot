# 💬 How to ask Coach Max so it answers

## TL;DR

Coach Max usually picks up your message just fine — but it occasionally misses messages that don't sound like clear questions. If Coach Max doesn't reply within ~5 minutes, **rephrase as a clear question** or **@-mention Coach Max** and it'll respond.

---

## ✅ What works — clear questions get answered

These phrasings reliably get Coach Max's attention:

| Style | Example |
|---|---|
| Direct question | *"What's the process for issuing an A&E refund?"* |
| "How do I..." | *"How do I issue a return label?"* |
| "Where do I find..." | *"Where do I find the SOP for chapter transfers?"* |
| "Can you tell me..." | *"Can you tell me the cutoff time for refunds?"* |
| Scenario + ask | *"A member is asking about X — what's the process?"* |
| @-mention | *"@Coach Max [anything]"* — almost always picked up |

The pattern: a real question word (*what / how / where / when / why / can / should*) anywhere in the message tells Coach Max "this is an ask."

---

## ⚠️ Phrasings that might get missed

These often look like statements, status updates, or announcements — not questions. Coach Max may pass on them. Easy fix: rephrase.

| What you might naturally write | Why Coach Max may skip it | Better way to ask |
|---|---|---|
| *"I'm having trouble with X"* | Sounds like a status update | *"How do I troubleshoot X?"* |
| *"X isn't working"* | Sounds like a complaint | *"What's the process when X isn't working?"* |
| *"Need help with X"* | Borderline — sometimes filtered | *"What's the SOP for X?"* |
| *"FYI a member..."* | Looks like an announcement | *"A member is asking about Y — what's the process?"* |
| *"Quick question — X"* (no question follows) | The "X" needs to actually be a question | *"Quick question — what's the SOP for X?"* |
| Just sharing context with no ask | Coach Max waits for an actual question | Add the question explicitly |
| *"Anyone know about X?"* | Coach Max sees this as directed at the team | *"What's the SOP for X?"* (directly to Coach Max) |

---

## 🛡 Two failsafes

If the wording is awkward or you just want to be sure:

1. **Use a question word** — *what / how / where / when / why / can / should* anywhere in your message
2. **@-mention Coach Max** — types `@` and pick Coach Max from the autocomplete; the highlighted blue tag is what catches the bot's attention

Either one almost always guarantees a response.

---

## 🤔 If Coach Max doesn't reply

1. **Wait ~5 min** — the bot wakes up every 5 minutes, so give it a tick to catch up
2. **Still nothing?** @-mention Coach Max and re-post the question with clearer wording
3. **Still no reply?** Ping Angelica — there may be something else going on (rare)

---

## 💡 Why this happens

Coach Max errs on the side of **staying quiet** when a message doesn't clearly look like a question — that's by design, so it doesn't spam the channel by replying to every status update, FYI, or compliment. The trade-off is that some clear questions get phrased in ways that look like statements.

You can think of it like: *Coach Max is polite. If you're not clearly asking it for something, it assumes you weren't talking to it.*

---

## 🔑 One thing to remember

**A clear question = a reliable answer.** The clearer your wording, the better Coach Max can both (a) decide to respond and (b) match it to the right SOP. When in doubt: phrase it as a question or tag it. 🙌
